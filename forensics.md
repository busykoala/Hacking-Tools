## Forensics

[Reference forensics tools](https://www.hackingarticles.in/forensics-tools-kali/)

### Binwalk

#### Installation
```
$ sudo pacman -S binwalk
```
#### Usage
```
$ binwalk file //show files in file
$ binwalk -e file //extract files in file
```

### Autopsy
